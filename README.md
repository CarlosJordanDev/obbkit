# Overview

obbkit is a CLI tool for generating boilerplates with onboardbase preconfigured in them as a SecOps infrastucture.

# Installation

### Go (package manager)
 
Install
 
```
go install 
```

Uninstall
```
```

### macOS

Install

```
$ brew tap adeboyedn/obbkit
$ brew install  adeboyedn/obbkit/obbkit
```

Update

```
brew upgrade obbkit
```

Uninstall

```
brew remove obbkit
```

### Version

```
obbkit --version
```
